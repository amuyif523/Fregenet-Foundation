## Partner entry example (content/site.md)
```yaml
partners:
  - name: "Bank Partner"
    logo: "/partners/bank-partner.svg"
    alt: "Bank Partner logo"
    descriptor: "Banking partner"
    href: "https://partner.example.com"
```

## Leader entry example (content/site.md)
```yaml
leaders:
  - name: "Leader Name"
    role: "President"
    bio: "Oversees governance and registrations."
    photo: "/photos/leader-name.jpg"
    photoAlt: "Portrait of Leader Name"
```

## Governance doc entry example (content/site.md)
```yaml
governanceDocs:
  - title: "Annual report 2024"
    href: "/docs/annual-report-2024.pdf"
    description: "Detailed financials and program reporting."
```

## Page frontmatter hero and images example
```yaml
heroImage: "/photos/hero.jpg"
heroImageAlt: "Students in Addis Ababa classroom"
heroImageCaption: "Students during morning lesson."
heroImageCredit: "Photo by Foundation Staff"
images:
  - src: "/photos/programs-addis.jpg"
    alt: "Teacher assisting students"
    caption: "Addis Ababa campus."
    credit: "Photo by Foundation Staff"
```
